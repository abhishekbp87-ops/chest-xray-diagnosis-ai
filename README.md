# Chest X-Ray Diagnosis AI

This repository contains the source code for a chest X-ray AI project. Large datasets, processed images, trained model weights, and local virtual environments are intentionally excluded from version control so the repository stays lightweight and can be pushed to GitHub reliably.

## Dataset

Download the chest X-ray dataset from Kaggle:

https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia

After downloading:

1. Extract the dataset locally.
2. Place the raw dataset under `data/`.
3. Keep generated outputs under `processed_data/`, `results/`, and `models/` on your machine only.

These folders are ignored by Git and should not be committed.

## Tracked in Git

This repository should only track source code, frontend assets such as HTML/CSS/JS, and lightweight configuration/documentation files.
